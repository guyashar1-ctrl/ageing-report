# Ageing Report – מערכת גיול חשבונות

מערכת Python+Streamlit לעיבוד קבצי מבנה אחיד (BKMVDATA.TXT) ודו"ח 331
ויצירת קובץ Excel עם נוסחאות SUM אמיתיות.

---

## ארכיטקטורה

```
ageing_report/
├── app.py                        # ממשק Streamlit (UI)
├── config/
│   ├── constants.py              # קבועים: סוגי רשומות, קידודים, שמות שדות
│   └── field_mappings.py         # מיפוי שדות לקובץ BKMVDATA (B100, C100)
├── parsers/
│   ├── bkmv_parser.py            # פרסור קבצי מבנה אחיד
│   └── report331_parser.py      # פרסור PDF דו"ח 331
├── processors/
│   └── balance_calculator.py    # אלגוריתם גיול ח/ז
├── excel/
│   └── excel_generator.py       # יצירת Excel עם נוסחאות
├── utils/
│   ├── file_utils.py             # חילוץ ZIP, איתור קבצים, קידוד
│   └── logger.py                 # לוגינג מרכזי
├── tests/
│   └── test_balance_calculator.py  # בדיקות יחידה לאלגוריתם הגיול
└── requirements.txt
```

## זרימת עבודה

1. **קבצי מבנה אחיד** – העלה קובץ ZIP **או** הזן נתיב לתיקייה המכילה
   `BKMVDATA.TXT` (או `TXT.BKMVDATA`).
2. **דו"ח 331** – העלה קובץ PDF.
3. **קוד כותרת** – הזן קוד (למשל `1342`).
4. לחץ **"אשר ועבד"**.
5. הורד את קובץ ה-Excel.

---

## מבנה קובץ ה-Excel

| גיליון | תוכן |
|--------|------|
| **ריכוז** | שורה אחת לכל חשבון: מספר, שם, יתרות, **נוסחת SUM**, תאריך תחילת חוב |
| **תנועות נבחרות** | התנועות שנבחרו לכל חשבון – מקור הנוסחה |
| **כל התנועות** | כל תנועות 2025 לכל חשבון (ביקורת) |
| **לוג שגיאות** | אזהרות ושגיאות מהעיבוד |

### נוסחת SUM

בגיליון **ריכוז**, עמודה E ("סכום ח/ז") מכילה נוסחה אמיתית:

```
=SUM('תנועות נבחרות'!D5:D12)
```

כך שהמשתמש יכול לפתוח את Excel ולראות את הנוסחה בשורת הנוסחאות.

## אלגוריתם הגיול

לכל חשבון:

1. **קביעת כיוון** – יתרת הסגירה חיובית (חובה)? שלילית (זכות)? אפס?
2. **טעינת תנועות** – כל תנועות שנת 2025 לחשבון זה.
3. **סינון צד רלוונטי** – תנועות חובה בלבד (אם יתרה חובה) / זכות בלבד.
4. **מיון** – ישן לחדש, ממוין יציב לפי (תאריך, מספר תנועה, מספר שורה).
5. **צבירה לאחור** – מהחדש ביותר לישן, עד שהסכום ≥ |יתרת סגירה|.
6. **תנועות נבחרות** – ה"זנב" של רשימת התנועות הממוינת.
7. **תאריך תחילת חוב** – תאריך התנועה הישנה ביותר שנבחרה.

## כיצד לשנות מיפוי שדות

פתח את `config/field_mappings.py`.

**דוגמה:** אם בקובץ שלך שם החשבון נמצא בעמודה 3 (במקום 2):

```python
B100_FIELDS = {
    ...
    "account_name": 3,   # שנה מ-2 ל-3
    ...
}
```

כל שאר הקוד קורא את הערכים דרך המיפוי הזה – אין צורך לשנות שום דבר אחר.

### פריסה חלופית – יתרה עם סימן (לא קוד צד)

אם הקובץ שלך מאחסן יתרה עם סימן (חיובי=חובה, שלילי=זכות) ולא
קוד נפרד עבור הצד, הפעל את הדגל:

```python
# config/field_mappings.py
USE_SIGNED_BALANCE_IN_B100 = True
```

## פתרון בעיות נפוצות

| בעיה | פתרון |
|------|-------|
| BKMVDATA.TXT לא נמצא | בדוק ש-ZIP מכיל את הקובץ; נסה שינוי שם |
