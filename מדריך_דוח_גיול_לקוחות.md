## שלב 3: חילוץ רשימת חשבונות מדוח 331

### 3.1 זיהוי סעיף 1342
סעיף **1342 - חובות פתוחים בארץ** הוא הסעיף הרלוונטי בדוח 331.

### 3.2 חילוץ מה-PDF
```python
import pdfplumber
import re

pdf_accounts = {}
with pdfplumber.open(pdf_path) as pdf:
    in_section = False
    for page in pdf.pages:
        text = page.extract_text()
        for line in text.split('\n'):
            # תחילת סעיף 1342 (טקסט בסדר ויזואלי)
            if 'ץראב םיחותפ תובוח 1342' in line and 'כ"הס' not in line:
                in_section = True
                continue
            # שורות המשך
            if in_section and '(ךשמה)' in line:
                continue
            # סוף סעיף 1342
            if in_section and '1342' in line and 'כ"הס' in line:
                in_section = False
                continue
            if in_section:
                # פורמט שורה: יתרה סוג קבוצה שם מספר_חשבון
                m = re.match(
                    r'^([\d,]+\.\d{2})\s+(ח|ז)\s+(\d+)\s+(.+?)\s+(\d{3,6})$',
                    line.strip()
                )
                if m:
                    acct_num = int(m.group(5))
                    pdf_accounts[acct_num] = {
                        'balance_pdf': float(m.group(1).replace(',', '')),
                        'bal_type': m.group(2)  # ח=חובה, ז=זכות
                    }
```

### 3.3 הערות
- הטקסט ב-PDF מחולץ בסדר ויזואלי (עברית הפוכה)
- `ח` = יתרת חובה (דביט), `ז` = יתרת זכות (קרדיט)
- מספר החשבון הוא המספר האחרון בשורה
- היתרה היא המספר הראשון בשורה

## שלב 4: חילוץ נתונים מהמבנה האחיד

### 4.1 קריאת B11 - יתרות פתיחה ושמות חשבונות
```python
b11_data = {}
with open('bkmv_data/BKMVDATA.TXT', 'rb') as f:
    for line in f:
        if not line.startswith(b'B11'):
            continue
        d = line.rstrip(b'\r\n\x85').decode('cp862', errors='replace')

        acct_num = int(d[22:37].strip())
        name_visual = d[37:87].strip()
        name_logical = name_visual[::-1]  # היפוך לסדר לוגי

        # חילוץ סכומים (סימן + 14 ספרות)
        amts = re.findall(r'[+-]\d{14}', d[270:])
        opening   = int(amts[0]) / 100  # יתרת פתיחה
        debits    = int(amts[1]) / 100  # סה"כ חובה
        credits   = int(amts[2]) / 100  # סה"כ זכות
        closing   = opening + debits - credits  # יתרת סגירה

        b11_data[acct_num] = {
            'name': name_logical,
            'opening_balance': opening,
            'closing_balance': closing
        }
```

### 4.2 קריאת B1 - תנועות יומן
```python
from collections import defaultdict
transactions = defaultdict(list)

with open('bkmv_data/BKMVDATA.TXT', 'rb') as f:
    for line in f:
        if not (line.startswith(b'B1') and not line.startswith(b'B11')):
            continue
        d = line.rstrip(b'\r\n\x85').decode('cp862', errors='replace')

        acct_num = int(d[172:187].strip())
        date_str = d[156:164]           # YYYYMMDD
        dc       = d[202:203].strip()   # 1=חובה, 2=זכות
        amt      = int(re.search(r'[+-]\d{14}', d[203:]).group()) / 100

        if dc in ('1', '2') and amt != 0:
            transactions[acct_num].append((date_str, dc, amt))
```

## שלב 5: חישוב תאריך תחילת חוב (גיול)

### 5.1 הלוגיקה
העיקרון: סוכמים תנועות **מסוף השנה לתחילתה**, רק תנועות מאותו סוג כמו היתרה, עד שהסכום המצטבר **עובר את יתרת הסגירה**.

#### עבור יתרת חובה (חיובית):
1. מסננים רק תנועות חובה (dc=1) עם סכום חיובי
2. ממיינים לפי תאריך **יורד** (מהסוף להתחלה)
3. סוכמים אחד אחד עד שהסכום המצטבר >= יתרת הסגירה
4. התאריך שבו עברנו את הסף = **תאריך תחילת החוב**

#### עבור יתרת זכות (שלילית):
- אותו הדבר, רק עם תנועות זכות (dc=2)
